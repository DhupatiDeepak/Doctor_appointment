# Supra Hospital - Doctor Appointment System

Supra Hospital is a premium, full-stack medical platform designed for seamless doctor appointments and healthcare management. Built with a modern tech stack, it features a responsive, elite-class UI with glassmorphic elements and smooth animations.

## 🚀 Features

- **Premium Responsive UI:** Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Homepage:** Features an animated hero slider, stats bar, and quick-access services.
- **Doctor Directory:** Search and filter specialists by category.
- **Appointment Booking:** Intuitive multi-step form for scheduling medical consultations.
- **Service Catalog:** Detailed overview of medical facilities and additional specialties.
- **Advanced Animations:** Specialized CSS `reveal` and `float` animations for a superior UX.
- **Glassmorphism:** Modern UI design using high-contrast glass elements.
- **Full-Stack Integration:** Robust Node/Express backend with MongoDB storage.

## 🛠️ Technology Stack

### Frontend
- **React 18** (Vite-powered)
- **Tailwind CSS v4** (Utility-first styling)
- **React Router Dom** (Navigation)
- **Lucide React** (Iconography)
- **Framer Motion** (Micro-interactions)
- **Axios** (API communication)

### Backend
- **Node.js & Express** (Server infrastructure)
- **MongoDB & Mongoose** (Database management)
- **JSON Web Token (JWT)** (Authentication)
- **Bcryptjs** (Security)
- **Multer & Cloudinary** (Media management)

## 📦 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)
- Git

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DhupatiDeepak/Doctor_appointment.git
   cd Doctor_appointment
   ```

2. **Backend Setup:**
   ```bash
   cd server
   npm install
   ```
   *Create a `.env` file in the `server` directory and add your MongoDB URI and port.*

3. **Frontend Setup:**
   ```bash
   cd ../client
   npm install
   ```

### Running the Application

1. **Start the Backend:**
   ```bash
   cd server
   npm start
   ```

2. **Start the Frontend:**
   ```bash
   cd client
   npm run dev
   ```
